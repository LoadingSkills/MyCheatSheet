HTML and CSS Helper

HTML Tags

Tag What it is What it does
a anchor allows a hyperlink "href" to be added to selected html
img Image inserts an image from the given "src" attribute
src opening tag attribute tells the browser where to get the IMAGE or AUDIO or VIDEO
href opening tag attribute tells the browser what webpage to link to when clicked
class opening tag attribute assigns a category or "class" to an element. Can make it easier to style
style CSS tag for HTML highly frowned upon code mixing swill. But yeah, you can code CSS in an HTML doc with this tag
link connector of files links your stylesheet and anything else to you html doc! Must be placed in the "head" section

CSS Properties

Property What it does
color sets the color of the TEXT
background-color sets the color of the box behind the text
background-image makes the box behind the text an image (if URL is set correctly)
margin sets the space between the border and other content. Set to auto for centering (like this table is)
display changes the behavior of content, it's borders, margins etc. Can be set to block, inline-block, etc.
text-align sets the alignment of TEXT in a given element. won't align images or borders
position sets where the content lives. Fixed, static, relative, sticky, absolute.
@font-face The devil's way of linking fonts into the CSS stylesheet. Just link it in the HTML you accursed tormentor
.class adding a period selects all elements belonging to the class specified
#id adding the number or pound sign selects the singular specific piece of HTML identified
commas or no commas to select multiple elements, no commas to grab a nested element
